## Get Started

1. clone repo
```md
HTTPS  : git clone https://github.com/theroom404/web-app.git

SSH    : git clone git@github.com:theroom404/web-app.git

GH-CLI : gh repo clone theroom404/web-app
```
2. `cd ./web-app`
3. `npm install`
4. `npm start`
