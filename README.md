## Get Started

1. clone repo
```md
HTTPS  : git clone https://github.com/theroom404/poke-store.git

SSH    : git clone git@github.com:theroom404/poke-store.git

GH-CLI : gh repo clone theroom404/poke-store
```
2. `cd ./poke-store`
3. `npm install`
4. `npm start`
